
## Definition
Array is a collection of elements stored in contiguous memory locations.

## Advantages
- Fast access
- Easy traversal

## Disadvantages
- Fixed size

## Time Complexity

| Operation | Complexity |
|---|---|
| Access | O(1) |
| Search | O(n) |
| Insert | O(n) |
| Delete | O(n) |

## Java Example

```java
int[] arr = {1,2,3,4,5};

for(int i : arr){
    System.out.println(i);
}
```
